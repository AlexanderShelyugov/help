## Configuration

```
"auth": {
    "backend": [
        "http://example.com/stalker_portal/server/api/chk_flussonic_tmp_link.php"
    ],
    "default": "allow"
}
```

## Portal Configuration

In the settings of the Ministra/Stalker turn on option "Temporary URL - Flussonic support"
